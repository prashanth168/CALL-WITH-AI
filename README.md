# 📞 Call with AI

A prototype system enabling users to interact with AI over simulated voice calls. Converts speech to text and responds with natural language.

- 💬 AI-generated replies

## 🛠️ Technologies
- Web Speech API / Whisper
- OpenAI GPT (Optional)
- React

```bash
